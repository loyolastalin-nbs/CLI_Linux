# Install Ansible on CentOS-7

## Run the following commands from root: (just use sudo su - )
================================

```
$ python --version

$ yum install  https://dl.fedoraproject.org/pub/epel...  (epel-release-latest-7.noarch.rpm)

$ yum install ansible

$ ansible --version

$ cd /etc/ansible/

$ ls -lrt

```
